Documentation

Directory structure

  • Administrations

    List of Canadian federal, provincial, territorial and municipal administrations. Used to lookup bilingual names.

  • code

    List of released open source code by administrations. Sub-folders contain data files. One file per administration.
    • federal
    • provincial
    • municipal

  • open_source_software

    List of open source software used by administrations.

How to add your data

  • Adding you administration

    • In the administrations directory add your adminstration in the federal.yml, provincial.yml or municipal.yml file.

  • Adding Open Source Code

    • Your administration is already there:

      If there is already a file for your adminstration under the code directory, you need to edit that file to add your released open source code.

      You can copy the contents of the file from the following example, and change the informtion for your code. These are only the mandatory fields. For more informations, click here to see the Metadata Schema for Open Source Code.

    • Your administration needs to be added:

      If your adminstration doesn't have a file under the code directory, you need to create a new file for your administration. Use the code template with only mandatory fields.
