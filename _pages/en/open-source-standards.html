Open Standards

Canadian public administrations' open standards.

See the {{ site.data.i18n.general.nav.doc[page.lang] }} and Open Standards Metadata Schema for the list of fields and how to add your data.

The data used to generate this list is hosted on GitHub. Create Pull Requests to add or update information on open standards used in Canadian federal, provincial, territorial and municipal administrations.

You can use the Open Standards Form to update existing data.

Data is also made available in JSON format for open data and use in other applications.
