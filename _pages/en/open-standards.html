Open Standards

Canadian public administrations' open standards.

The data used to generate this list is hosted on GitHub. Create Pull Requests to add or update information on open standards used in Canadian federal, provincial, territorial and municipal administrations. Data is also available in JSON format.

You can use the Open Standards Form to update existing data.

Data is also made available in JSON format for open data and use in other applications.
