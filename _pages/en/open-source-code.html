Open Source Code

The data used to generate this list is hosted on GitHub. Create Pull Requests to add or update information on Canadian federal, provincial, territorial and municipal administrations Open Source Code. Data is also available in JSON format.

See the list of Open Source Code Groups to find more Canadian public administrations' open source projects and code repositories.

If you are interested to contribute with us, fill out the open source code form. The metadata schema can also help you to validate the form.
