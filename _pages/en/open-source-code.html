Open Source Code

Canadian public administrations' open source projects and code repositories.

See the list of Open Source Code Groups to find more Canadian public administrations' open source projects and code repositories.

See the Documentation and Open Source Code Metadata Schema for the list of fields and how to add you data.

{% include codeTable.html %}

The data used to generate this list is hosted on GitHub. Create Pull Requests to add or update information on Canadian federal, provincial, territorial and municipal administrations Open Source Code.

You can use the Open Source Code Form to update existing data.

Data is also made available in JSON format for open data and use in other applications.
