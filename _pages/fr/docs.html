Comment utiliser la plateforme Échange de Ressources Ouvert

La plateforme Échange de Ressources Ouvert offre trois services basés sur le partage de solutions numériques ouvertes en provenance des différentes administrations publiques canadiennes. Vous trouverez ici des guides d’utilisation relatifs à chaque service offert par les différentes sections de la plateforme.

  • Codes Sources Ouverts

    Ce service offre une plateforme de partage des différents codes sources ouverts développés par des administrations publiques canadiennes fédérales, provinciales, municipales et autochtones.

    Vous y trouverez un catalogue (ou une liste autrement dit) contenant tous les projets libres disponibles. Chaque code source est brièvement décrit par les champs suivants:

    • Nom du projet: Le nom du projet correspond au nom du code source développé en libre. L'hyperlien permet d’afficher des informations relatives au projet, tels les informations de l’administration publique ayant développé le projet, la date de création, la dernière mise à jour, etc.
    • Catégorie: Il s’agit de la catégorie à laquelle appartient le code source ouvert en ce qui concerne son domaine d’utilisation. Les catégories proposées correspondent à la famille des logiciels de la taxonomie UNSPSC.
    • Description: Il s’agit d’une brève description des fonctionnalités du programme.
    • Licence : C’est le type de licence sous laquelle le projet est publié. L’hyperlien permet d’accéder au fichier de la licence du projet, qui est normalement dans le répertoire public où le code est hébergé (par exemple dans GitHub).
    • Code repo: C’est le répertoire public du code source, où le code est hébergé (par exemple GitHub, GitLab, BitBucket, etc.). L’hyperlien permet d’accéder au dossier du projet déposé dans le répertoire public.

    Utilisez ce catalogue pour trouver des normes ouvertes qui peuvent vous aider à développer un système qui fonctionne et communique avec d’autres produits ou systèmes, et qui peut être facilement mis à niveau et étendu. Afin de faciliter la recherche, plusieurs options de filtrage sont à votre disposition:

    • Vous pouvez filtrer les différents codes selon des options de filtrages, tels le niveau de gouvernement, l’administration publique, le mot-clé, etc.
    • Vous pouvez également filtrer les noms et les descriptions des codes sources disponibles dans le catalogue selon une phrase, un mot ou une lettre de votre choix.

    Vous désirez ajouter un nouveau projet ou mettre à jour les informations d’un projet déjà disponible dans le catalogue?

    Que vous soyez une administration ayant déjà contribué à cette plateforme ou une administration désirant participer pour la première fois à la contribution de celle-ci, vous êtes libre d’ajouter une nouvelle solution au catalogue ou de mettre à jour les informations d’un projet déjà existant. Il vous suffit de remplir le formulaire Formulaire de code source ouvert permettant d’identifier votre requête et d’exécuter celle-ci. Remplissez les champs s’alignant avec votre requête et soumettez le formulaire.

  • Logiciels Libres

    Ce service offre une plateforme de partage des différents logiciels libres utilisés par des administrations publiques canadiennes fédérales, provinciales, municipales et autochtones.

    Vous y trouverez un catalogue (ou une liste autrement dit) contenant tous les logiciels libres disponibles. Ceux-ci sont brièvement décrits par les champs suivants:

    • Logiciel: Ce champ correspond au nom du logiciel libre. L'hyperlien permet d’afficher une liste des administrations utilisant le logiciel.
    • Catégorie: Il s’agit de la catégorie à laquelle appartient le logiciel en ce qui concerne son domaine d’utilisation. Les catégories proposées correspondent à la famille des logiciels de la taxonomie UNSPSC.
    • Description: Il s’agit d’une brève description des fonctionnalités du programme.
    • Licence : C’est le type de licence sous laquelle le projet est publié. L’hyperlien permet d’accéder au fichier de la licence du projet, qui est normalement dans le répertoire public où le code est hébergé (par exemple dans GitHub).
    • Usage: L’usage correspond au nombre d’administrations ayant validé leur utilisation du programme.

    Utilisez ce catalogue pour trouver une solution numérique qui répond à vos besoins. Afin de faciliter la recherche, plusieurs options de filtrage sont à votre disposition:

    • Vous pouvez filtrer les différents logiciels selon différentes options de filtrages, soit le niveau de gouvernement, la licence et le tag.
    • Vous pouvez également filtrer les noms et les descriptions des logiciels libres présentés dans le catalogue selon une phrase, un mot ou une lettre de votre choix.

    Vous désirez ajouter un nouveau logiciel libre au catalogue ou vous inscrire à la liste des utilisateurs de l’une des solutions partagées?

    Que vous soyez une administration ayant déjà contribué à cette plateforme ou une administration désirant participer pour le première fois à la contribution de celle-ci, vous êtes libre d’ajouter une nouvelle solution au catalogue ou de vous inscrire à la liste des utilisateurs de l’un des logiciels libres déjà partagés. Il vous suffit de remplir le formulaire Formulaire du Logiciel Libre permettant d’identifier votre requête et d’exécuter celle-ci. Remplissez les champs s’alignant avec votre requête et soumettez le formulaire.

  • Normes Ouvertes

    Ce service offre une plateforme de partage des différentes normes ouvertes utilisées par des administrations publiques canadiennes fédérales, provinciales, municipales et autochtones.

    Vous y trouverez un catalogue (ou une liste autrement dit) contenant toutes les normes partagées. Celles-ci sont brièvement décrites par les champs suivants:

    • Norme: Ce champs correspond au nom de la norme. L'hyperlien permet d’afficher une liste des administrations utilisant la norme en question.
    • Description: Il s’agit d’une brève description de la norme.
    • Organisation normative: Il s’agit de l’organisation ayant développé, coordonné, promulgué, révisé, modifié, interprété, publié ou produit la norme. L’hyperlien renvoie vers la spécification de licence.

    Utilisez ce catalogue pour trouver une norme qui répond à vos besoins. Afin de faciliter la recherche, plusieurs options de filtrage sont à votre disposition:

    • Vous pouvez filtrer les différentes normes selon le niveau de gouvernement et/ou le tag.
    • Vous pouvez également filtrer les noms et les descriptions des standards présentés dans le catalogue selon une phrase, un mot ou une lettre de votre choix.

    Vous désirez ajouter une nouvelle norme ou mettre à jour les informations d’une norme déjà présente dans le catalogue?

    Que vous soyez une administration ayant déjà contribué à cette plateforme ou une administration désirant participer à la contribution de celle-ci, vous êtes libre d’ajouter une nouvelle norme au catalogue ou de mettre à jour les informations de l’un des normes disponibles. Il vous suffit de remplir le formulaire Formulaire des Normes Ouvertes permettant d’identifier votre requête et d’exécuter celle-ci. Remplissez les champs s’alignant avec votre requête et soumettez le formulaire.
