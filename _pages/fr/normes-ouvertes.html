Normes ouvertes

Standards libres des administrations publiques Canadiennes.

Consultez la Documentation et le Schéma de métadonnées des normes ouvertes pour la liste des champs et comment ajouter vos données.

{% include standardsTable.html %}

Les données utilisées pour générer cette liste est hébergé sur GitHub. Créez des Pull Requests pour ajouter ou mettre à jour de l'information sur les normes ouvertes utilisés par les administrations fédérale, provinciales, territoriales et municipales du Canada.

Vous pouvez utiliser le Formulaire des normes ouvertes pour mettre à jours des données existantes.

Les données sont aussi disponible en format JSON.
