Code source ouvert

Les données utilisées pour générer cette liste est hébergé sur GitHub. Créez des Pull Requests pour ajouter ou mettre à jour de l'information sur le code source ouvert des administrations fédérale, provinciales, territoriales et municipales du Canada. Les données sont aussi disponible en format JSON.

Vous pouvez aussi consulter la liste des Groupes de code source ouvert pour trouver plus de projets libres et dépôts de code d'administrations publiques Canadiennes.

Si vous êtes intéressé à contribuer avec nous, veuillez remplir le formulaire de code source ouvert. Le schéma de métadonnées peut également vous aider à valider le formulaire.
