Code source ouvert

Projets libres et dépôts de code des administrations publiques Canadiennes.

Consultez la liste des Groupes de code source ouvert pour trouver plus de projets libres et dépôts de code source ouvert d'administrations publiques Canadiennes.

{% include codeTable.html %}

Les données utilisées pour générer cette liste sont hébergé sur GitHub. Créez des Pull Requests pour ajouter ou mettre à jour de l'information sur le code source ouvert des administrations fédérale, provinciales, territoriales et municipales du Canada.

Vous pouvez utiliser le Formulaire de code source ouvert pour mettre à jours des données existantes.

Les données sont aussi rendu disponible en format JSON pour utilisation dans d'autres applications.
