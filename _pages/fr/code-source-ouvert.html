Code Source Ouvert

Le code source ouvert fait référence au logiciel dont le développement suit un processus ouvert. La licence du projet permet d’accéder librement au code source, de créer des travaux dérivés et de le redistribuer. Ce processus de développement ouvert est généralement le résultat de la collaboration entre plusieurs parties prenantes pour l’élaboration de solutions technologiques à des problèmes communs.

Projets

Consultez la liste des différents projets ouverts élaborés par des administrations publiques canadiennes du niveau fédéral, provincial, municipal et autochtone. Vous pouvez modifier les informations d’un projet existant dans la liste ou ajouter un nouveau projet ouvert en soumettant le Formulaire de Code Source Ouvert.
Besoin d’aide? Consulter la documentation comment utiliser la plateforme Échange de Ressources Ouvert pour plus d’informations sur comment bénéficier de ce service.
