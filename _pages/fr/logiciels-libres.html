Logiciels libres

Logiciels libres des administrations publiques Canadiennes.

Consultez la Documentation et le Schéma de métadonnées de logiciels libres ouverts pour la liste des champs et comment ajouter vos données.

{% include ossTable.html %}

Les données utilisées pour générer cette liste sont hébergé sur GitHub. Créez des Pull Requests pour ajouter ou mettre à jour de l'information sur les logiciels ouverts des administrations du Canada.

Vous pouvez utiliser le Formulaire de logiciels libres ouverts pour mettre à jours des données existantes.

Les données sont aussi rendu disponible en format JSON pour utilisation dans d'autres applications.
